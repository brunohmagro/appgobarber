## Building :arrow_forward:

You'll need [Node.js](https://nodejs.org) and run the Backend on your computer in order to build this app.

```
git clone git@github.com:brunohmagro/appgobarber.git
$ cd appgobarber
$ yarn install
$ yarn android or yarn ios
```
